================================================================
JOB AGENT — HOW TO USE
================================================================

WHAT THIS PROJECT DOES
-----------------------
An automated job search agent that runs daily and:
  1. Scrapes remote job listings from RemoteOK and Indeed
  2. Scores each job 0-10 based on how well it matches your profile
  3. Tailors your resume for the top-scoring jobs (score >= 7)
  4. Generates a daily digest summarising all opportunities

----------------------------------------------------------------

BEFORE YOU RUN — ONE-TIME SETUP
---------------------------------

  Step 1: Add your master resume
  --------------------------------
  Create a file:  job-agent/resumes/resume.txt

  Write your resume in plain text. Example format:

    ## Summary
    Software engineer with 3 years experience in Python and AWS.

    ## Experience
    ### Backend Engineer @ Acme Corp (2022-2025)
    - Built REST APIs using FastAPI serving 1M requests/day
    - Reduced query time by 40% via PostgreSQL optimisation

    ## Skills
    Python, FastAPI, Django, PostgreSQL, AWS, Docker, Git

    ## Education
    BSc Computer Science, University of XYZ, 2021

  The more detail you add, the better the tailored CVs will be.

  Step 2: Configure your job search
  ------------------------------------
  Open job-agent/.env and set:

    ANTHROPIC_API_KEY=sk-ant-...      ← already set
    JOB_TITLES=Software Engineer,Backend Engineer,Python Developer
    JOB_LOCATION=Remote

  Step 3: Update your candidate profile
  ----------------------------------------
  Open job-agent/crew.py and edit CANDIDATE_PROFILE (line ~14).
  Write a short paragraph about yourself:

    "5 years Python experience, Django and FastAPI, AWS certified,
     looking for senior remote backend roles."

HOW TO RUN main.py
--------------------

  1. Open Terminal (press Cmd + Space, type "Terminal", hit Enter)

  2. Navigate to the project folder:

       cd /Users/s.joon/Desktop/job_agent/job-agent

     (Type this exactly and press Enter)

  3. Run the agent:

     To run ONCE (recommended for testing):

       python3 main.py

     To run on a DAILY SCHEDULE at 8am (keep terminal open):

       python3 main.py --schedule

  4. Watch the terminal output:

       [crew] Scraping jobs...
       [scraper] Searching: Software Engineer    ← fetching listings
       [crew] 15 new jobs saved.
       [crew] Scoring 15 jobs...                ← Claude AI scoring
       [crew] Tailoring CV for: ...             ← generating resumes
       [crew] Writing digest...
       [crew] Digest: outputs/digest_2026-03-17.md

  5. When finished, check the outputs/ folder:

       /Users/s.joon/Desktop/job_agent/job-agent/outputs/

     You will find:
       - digest_YYYY-MM-DD.md   → open in any text editor
       - CV_Company_Role.docx   → open in Word or Google Docs

----------------------------------------------------------------

WHAT TO DO EVERY DAY
----------------------
  1. Open Terminal and run:   python3 main.py
  2. Open outputs/digest_<today>.md — read the top picks
  3. For any job you like, open the matching CV_*.docx
  4. Review and edit the tailored CV if needed
  5. Apply using the job URL listed in the digest

----------------------------------------------------------------

TROUBLESHOOTING
----------------
Problem: "No new jobs to process"
  → All jobs are already in the database from a previous run
  → Reset by deleting the database:
      rm /Users/s.joon/Desktop/job_agent/job-agent/data/jobs.db
  → Then run again: python3 main.py

Problem: "Add a master resume to resumes/"
  → Create the file: job-agent/resumes/resume.txt
  → See "Before You Run" section above

Problem: AuthenticationError (401)
  → Your Anthropic API key is wrong or expired
  → Get a new key at: console.anthropic.com
  → Update it in: job-agent/.env

Problem: SSL warning about LibreSSL
  → Harmless macOS warning — ignore it, everything still works

Problem: "command not found: python3"
  → Python is not installed
  → Download from: python.org/downloads

Problem: No jobs found / very few results
  → Indeed blocks scrapers sometimes — RemoteOK will still work
  → Try again later or adjust JOB_TITLES in .env
